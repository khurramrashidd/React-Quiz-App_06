
# 🧠 React Quiz App (Day 6)

A modern, interactive **Quiz Application built using React** that tests your knowledge and calculates your score dynamically.

This project demonstrates core React concepts like **conditional rendering, mapping through data arrays, and complex state management**.

This project is part of my **React learning journey (Day 6 Project)**.

---

## 🚀 Features

* Dynamic question rendering from a separate data file
* Real-time score tracking
* Conditional rendering (Quiz Screen vs. Result Screen)
* Instant feedback on quiz completion
* Restart quiz functionality
* Responsive and clean UI
* Component-based architecture (Header & Footer)

---

## 🛠 Built With

* **React (Vite)**
* JavaScript (ES6+)
* CSS (Responsive Design)
